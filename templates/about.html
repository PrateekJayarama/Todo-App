About the Todo App 📝

Welcome to the Todo App, your ultimate task manager designed to help you stay organized and productive! Whether you need to track daily tasks, set reminders, or update your to-dos on the go, this app has you covered.

🌟 Why Use This App?

  • Simple & User-Friendly – A clean, intuitive interface for effortless task management.
  • Quick Task Updates – Edit your to-dos anytime, anywhere.
  • Efficient Task Tracking – View and manage all tasks in one place.
  • Secure & Fast – Built using Flask, ensuring smooth performance.

🚀 Features at a Glance

  • 📌 Create Tasks – Add new tasks easily.
  • 📌 View Your Todos – Access a list of all your to-dos.
  • 📌 Edit & Update – Modify task details anytime.
  • 📌 Delete Tasks – Remove completed or unnecessary tasks.

🔧 Technology Stack

  • 💻 Flask – Backend framework for routing and logic.
  • 🎨 HTML, CSS – Clean and responsive UI design.
  • 🛠 Jinja2 – Dynamic HTML templating.
  • 📦 Database – SQLite.
